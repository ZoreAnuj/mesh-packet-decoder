# Mesh Packet Decoder

A TypeScript library for decoding MeshCore mesh networking packets with full cryptographic support. This project provides a foundational tool for analyzing and understanding the packet structure and security protocols within mesh networks, enabling deeper exploration of the domain.

## Key Features
*   Decodes MeshCore packet headers and payload structures.
*   Supports cryptographic verification of packet integrity and authenticity.
*   Provides a clean, type-safe API for programmatic packet inspection.
*   Includes utilities for parsing common mesh network data formats.

## Tech Stack
*   TypeScript
*   Node.js

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/mesh-packet-decoder.git`
2.  Install dependencies: `npm install`
3.  Build the library: `npm run build`
